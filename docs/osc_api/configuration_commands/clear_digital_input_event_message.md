# Clear Digital Input Event Message

```
/sio/cfg/osc/reportMsg/di/{channel}/{state}/clear
```

...where {channel} is a single character channel number and {state} is either "hi" or "lo".

Clears the custom OSC message for the specified digital input channel and state.

### Arguments

None

## Effects
Removes the custom event message from the device configuration for the specified channel and state.
