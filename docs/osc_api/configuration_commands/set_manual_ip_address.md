# Set Manual IP Address

```
/sio/cfg/lan/manual/ip/set
```

Sets the static IP address for Manual IP mode.

### Arguments

- Arg 0 (`int` | `float`): First octet (0-255)
- Arg 1 (`int` | `float`): Second octet (0-255)
- Arg 2 (`int` | `float`): Third octet (0-255)
- Arg 3 (`int` | `float`): Fourth octet (0-255)

## Effects
Updates the Manual IP address configuration.
