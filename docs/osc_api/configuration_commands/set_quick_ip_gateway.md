# Set Quick IP Gateway

```
/sio/cfg/lan/quick/gateway/set
```

Sets the gateway IP address for Quick IP mode.

### Arguments

- Arg 0 (`int` | `float`): First octet (0-255)
- Arg 1 (`int` | `float`): Second octet (0-255)
- Arg 2 (`int` | `float`): Third octet (0-255)
- Arg 3 (`int` | `float`): Fourth octet (0-255)

## Effects
Updates the Quick IP gateway configuration.
