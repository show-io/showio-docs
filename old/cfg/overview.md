# Configuration API
OSC Methods in the ```/sio/cfg/``` branch allow the user to configure a ShowIO device.

| Method                                                                | Address                                       |
| ------------------------------------------------------------------    | -------------------------------------------   |
| [Get Device Config](sio_cfg_get.md)                                   | ```/sio/cfg/get```                            |
| [Set Digital Input Event Message](sio_cfg_eventmessage_di_set.md)     | ```/sio/cfg/eventmsg/di/{channel}/set```      |
