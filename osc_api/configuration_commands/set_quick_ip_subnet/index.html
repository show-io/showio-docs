    Set Quick IP Subnet

    /sio/cfg/lan/quick/subnet/set

    Sets the subnet mask prefix length for Quick IP mode.

    Arguments

    • Arg 0 (int | float): Subnet mask prefix length (0-32)

    Effects

    Updates the Quick IP subnet mask configuration.
